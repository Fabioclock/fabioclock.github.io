# Support boucles

Petit support visuel pour aider à la découverte des boucles en cours. Une petite intro est disponible pour chacune, avec un système d'exécution pas à pas et des tooltips en passant au dessus du code pour constater l'évolution des données.

## Mode d'emploi

L'écran est divisé en deux parties. A gauche, une petite intro avec un exemple de boucle simple. Les différents éléments qui composent le code peuvent faire apparaître des infos supplémentaires au survol. 

A droite, un "écran de résultat". Il sert à afficher les actions effectuées par les boucles. Trois boutons permettent de jouer sur l'état de la boucle en l'exécutant étape par étape, tout d'un coup ou en la réinitialisant.

A chaque itération de la boucle, le résultat s'affiche sur l'écran de droite tandis que les variables dont le contenu change passent en surbrillance à gauche. Un survol permet de voir leur contenu. 

### Boucles concernées 

- For (PHP, JS)
- While (PHP, JS)
- Foreach (PHP)
- For...in (JS)
- For...of (JS)
